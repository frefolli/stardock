# Stardock

Directory Synchronizer

```
stardock [options]

Options:
    -h/--help         print help (this)
    -r/--refresh      always refreshes indexes
    -v/--verbose      uses verbose log
    -s/--src          specify src path
    -d/--dst          specify dst path
    -A/--asym         do asymmetric diff (src --> dst)
    -S/--sym          do symmetric diff (src <-> dst)
    -T/--transfer     do transfer (according to diff)
    -I/--index        do index (preventive for transfer and diff)
```
